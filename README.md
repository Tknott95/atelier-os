# atelier-os

This is a Rust package crate scaffolded for the `atelier-os` codebase.

## What is included

- `Cargo.toml` defining the Rust package metadata.
- `src/lib.rs` with a small exported `AtelierOs` type to get started.

## Quick start

Build the crate:

```bash
cargo build
```

Run a quick smoke test in Rust (or your own tests) using:

```bash
cargo test
```
